🎯 About This Quiz

📝 Quiz Format

Each quiz presents 8 randomly selected regex expressions from our database. For each expression, you'll see both the original regex and a refactored version, creating 16 total questions per quiz.

🎲 Randomization

Test strings are randomly selected with a 50% probability of being a correct match and 50% probability of being an incorrect match. This ensures balanced difficulty and prevents pattern memorization.

💾 Data Storage

All quiz results are automatically saved to our MongoDB database, allowing us to track performance trends and provide meaningful statistics to help improve learning.

🎓 Learning Objectives

  • Understand how regex patterns match against strings
  • Recognize the difference between original and refactored regex expressions
  • Develop intuition for regex pattern matching
  • Practice regex comprehension in a gamified environment
